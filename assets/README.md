Place your profile image here so the static site will display it.

- Filename: `photo.jpg`
- Recommended size: 800×800 (square) or similar; the CSS will crop/scale.
- Path (static): `assets/photo.jpg`

If you are using the React dev app (Vite), put the same image in `client/public/assets/photo.jpg` so it's served at `/assets/photo.jpg`.

After adding the image, commit and push it to GitHub, or run the dev server locally to verify.
